# Cybersecurity Task: Simple Network Sniffer

## 📝 Overview

This project is a basic packet sniffer developed in Python. It captures network traffic in real-time, allowing users to view data packets being transmitted over their local network interface.

## 🛡️ Key Features

* **Packet Capture:** Monitors and captures live network traffic.
* **Protocol Identification:** Identifies headers for protocols like IP, TCP, and UDP.
* **Data Display:** Displays source/destination IP addresses and port numbers.

## 🛠️ Tools & Technologies Used

* **Library:** `Scapy` (The industry-standard Python library for packet manipulation).
* **Language:** Python 3.x.
* **Environment:** Compatible with Linux (recommended for sniffing) or Windows/macOS.

## 🚀 How to Use

1. **Install Scapy:**
```bash
pip install scapy

```


2. **Run the script:**
*Note: Sniffing requires administrative or root privileges to access the network card.*
```bash
python network_sniffer.py

```
